Для каждого пользователя на сервере создается SQLite база данных (один файл).
Пользователь может делать с этой базой что угодно: создавать и удалять таблицы, добавлять данные, делать любые выборки... 

Это тестовая версия. Доступен только один пользователь и соответственно, одна база данных.
Запросы доступны любые, но чтобы сервер не завис, ограничен максимальный размер выборки 100x100.
